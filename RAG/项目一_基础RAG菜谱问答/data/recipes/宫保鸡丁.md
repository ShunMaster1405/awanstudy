# 宫保鸡丁的做法
宫保鸡丁是一道经典的川菜，以鸡肉丁、花生米和干辣椒为主要原料。
预估烹饪难度：★★★

## 必备原料和工具
* 鸡胸肉 300克
* 花生米 100克
* 干辣椒 10个
* 葱、姜、蒜适量
* 食用油、盐、糖、醋、酱油、料酒

## 计算
* 鸡胸肉 = 300克
* 花生米 = 100克
* 干辣椒 = 10个

## 操作
1. 鸡胸肉切丁，用料酒、盐腌制10分钟
2. 花生米用油炸至金黄酥脆
3. 干辣椒剪成小段，葱姜蒜切末
4. 热锅凉油，下鸡丁滑炒至变色
5. 加入干辣椒、葱姜蒜爆香
6. 加入调味料翻炒均匀
7. 最后加入花生米翻炒均匀即可

## 附加内容
可以根据个人口味调整辣度和酸甜比例。
